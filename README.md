# Notoria ☁️ – Write on Clouds

A glass-styled cloud note-taking app built with **FastAPI + MongoDB Atlas**.

## Features
- Add & edit notes
- Mark notes as important
- Glass-morphism UI
- MongoDB cloud storage

## Tech Stack
- FastAPI
- MongoDB Atlas
- Jinja2
- Bootstrap 5

## Installation

```bash
git clone https://github.com/nayan-ng/notoria.git
cd notoria
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
